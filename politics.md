# 🗳️ Politics Dashboard Analysis

## 📌 Project Overview
This project analyzes political and election-related data to understand voting patterns, party performance, and regional insights using an interactive Power BI dashboard.

## 🎯 Objectives
- Analyze vote share by political parties
- Compare performance across states/regions
- Identify winning patterns and trends
- Provide data-driven political insights

## 🛠 Tools Used
- Power BI
- Excel
- SQL (optional)

## 📊 Key Insights
- Leading party dominates in multiple regions
- Regional parties perform strongly in specific states
- Urban vs rural voting patterns vary significantly
- Certain regions show consistent voting behavior

## 📈 Dashboard Features
- Party-wise vote share visualization
- Region/state-wise performance
- Interactive slicers and filters
- KPI cards for quick insights
